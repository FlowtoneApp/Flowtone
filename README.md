# Flowtone

## Flowtone 由 AI 编写，人工极少量介入。

Flowtone 是一款 Android 音乐播放器，旨在提供舒适、稳定的本地音乐播放体验。

此项目正处于早期开发阶段。

当前最新 tag 版本为`0.8.1`。

当前最新 Pre-Release 版本为 `0.7.1`。  
如果你希望体验相对稳定的版本，建议下载 Pre-Release 版本；如果你希望查看最新开发进度，可以使用 `main` 分支。

请注意，`main` 分支可能包含尚未充分验证的开发改动。

未来，Flowtone 将逐步探索扩展生态，以拥抱开源音乐库与音乐记录社区。
